# 📖 Overview

FocusForge is a complete productivity ecosystem built for students.

It combines task management, habit tracking, goal management, journaling, analytics, rewards, Pomodoro focus sessions, and a Chrome extension that blocks distracting websites during study sessions.

The project uses Supabase for authentication, database storage, realtime synchronization, and secure backend services.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/NaikPoorvika/FocusForge.git
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Production build

```bash
npm run build
```

---

# 🔐 Environment Variables

Create a `.env` file using `.env.example`

Example:

```env
VITE_SUPABASE_URL=your_url
VITE_SUPABASE_ANON_KEY=your_key
```
